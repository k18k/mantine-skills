# mantine-skills

Mantine skill for LLMs.

This repository generates a `mantine` skill from Mantine's official LLM documentation.

## Install

```bash
npx skills add k18k/mantine-skills --skill mantine
````

## Generate skill

```bash
npm start
```

This will fetch:

[https://mantine.dev/llms.txt](https://mantine.dev/llms.txt)

and generate:

```
skills/mantine/SKILL.md
```

## Purpose

Use this skill when working with Mantine:

* components
* hooks
* forms (`@mantine/form`)
* theming
* styling
* API usage
